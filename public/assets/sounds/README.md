# Sound Assets for Chaos Genome

Add the following audio files to this directory:

## Ambient Loops
- `chaos_ambience_loop.mp3` - Low-frequency hacker lab background with:
  - Server fan noise
  - Electrical hum
  - Keyboard typing (ambient)
  - Recommended: 3-5 second loop, MP3 format, ~128 kbps

- `night_ambience_loop.mp3` - Peaceful night atmosphere (if not already present)

## Effect Sounds
- `glitch_boot.mp3` - System boot/glitch startup sound (~0.5 seconds)
- `gremlin_giggle.mp3` - Playful chaotic giggle sound (~0.8 seconds)
- `owl_hoot.mp3` - Night Owl creature sound (if not already present)

## Chaos Builder Scene Specific
- `error_beep.mp3` - Terminal error sound (0.2 seconds)
- `glitch_distortion.mp3` - Digital glitch/corruption sound (0.3 seconds)
- `wire_spark.mp3` - Electrical spark sound (0.4 seconds)

## Notes
- All audio should be MP3 format for broader browser compatibility
- Keep ambient loops around 128-192 kbps to reduce file size
- Effect sounds can be mono, ambient loops can be stereo
- Consider adding subtle variations for looping sounds to avoid repetition
